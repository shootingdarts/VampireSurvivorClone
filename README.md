# VampireSurvivorsClone
## Summer 2023 Project
### Game Description
- A different take on Vampire Survivors.
- With the lore centered around fighting back fungal infestation.
- Made with pygame.
- With help on game design and art from [littlepebbles](https://github.com/littlepebbles)
### Techniques
-	Utilizes a camera system that locks onto the player.
-	Projectiles from weapons have pixel-perfect hitbox using Pygame masks.
-	Player engagement is encouraged by its roguelike experience, score tracker, and scaling enemies
